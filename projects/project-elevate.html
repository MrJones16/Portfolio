    Escape From Project E.L.E.V.A.T.E.

    A grapple-hook based platformer built in Godot 4

    Explore the mysterious remains of project E.L.E.V.A.T.E.

    STORY 
    # Main character: Test Subject HOPE
# Mission: Traverse through the remains of E.L.E.V.A.T.E.
# Core mechanic: Grapple gun physics

    > ....Rebooting System for Model Name: HOPE, System ID: CM17PM2S2023...

    > ... Running Damage Analysis... Walk System Failure. Changed to Hover Support System

    > ... Grapple Tether in Functioning Condition.

    > "Where am I? Where is everyone else?"

    > ....Allow me to inform you HOPE. Project E.L.E.V.A.T.E. had an accident 3 years ago. All of the other robots were put out of commission. The facility you are trapped in is the lowest area of E.L.E.V.A.T.E., which the public doesn't know. Traverse through the area to find a way out of here. The answer should lie there.

    GAME_FEATURES

    MECHANICS
    • >Grapple gun traversal
    • >Green Zip Blocks for instant movement
    • >Jump Pads for vertical traversal
    CONTROLS
    • >W/SHIFT - Retract/Boost Retract
    • >S/CTRL - Extend/Boost Extend
    • >A/D - Left/Right movement
    • >Left Click - Grapple towards mouse
    • >Space/Right Click - Release grapple
    ABOUT THE VERTICAL SLICE

    The Compact Edition of this game focuses on experiencing the main gameplay loop, so the narrative is abbreviated and not included. The edition contains 10 levels you can choose from.

    Beating Level 10 will get you to a victory screen before returning back to the main menu.

    INSTALLATION

    DOWNLOAD_INSTRUCTIONS
    1. Download the zip file and extract the contents in another folder
    2. Click on the executable
    3. If windows blocks you from running the program, click "Run Anyway"
    4. The Game should be up and running :D
    5. Click "Play" to select a level!
