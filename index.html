Pyxis Game Engine

Skills:

  • C++
  • Engine Programming
  • Graphics Programming / GLSL

Learning to make a game engine in C++! I've been following tutorials from The Cherno's game engine series on youtube in order to learn game engine development. I hope to create a 2D or Voxel game engine that is pixel art themed, like Aseprite is. I currently have a Logging system using spdlog, an Event system, Window and Layers system, ImGui for rendering, and I am currently working on abstracting rendering into my own API so that the engine can use different Renderer API's than just OpenGL, like DirectX12. If you are interested, I have more information about what it uses and what I've learned on the github. Just click the link below!
